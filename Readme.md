# Superstore Data Analysis Dashboard

This repository contains resources for a Power BI-based analysis of Superstore sales data. It includes the dataset and the Power BI dashboard file used to create interactive visualizations and insights.

## 📁 Repository Contents

- **`Superstore Data.xlsx`** (or similar) – The raw sales data file containing transaction records for a fictional superstore. This dataset includes fields such as Order Date, Sales, Profit, Customer Segment, Product Category, and more.
- **`Superstore Dashboard.pbix`** – Power BI Desktop file with reports and dashboards built from the dataset. It features charts, maps, and key performance indicators (KPIs) for exploring sales trends, regional performance, and customer behavior.

> ⚠️ If the file names differ, please adjust accordingly.

## 🎯 Project Overview

The goal of this project is to perform exploratory data analysis (EDA) and generate actionable insights through an interactive Power BI dashboard. By visualizing the Superstore data, stakeholders can:

1. Track overall sales and profit trends over time.
2. Identify high-performing regions and store segments.
3. Examine product category performance and top-selling items.
4. Analyze customer segments and order priorities.
5. Support data-driven decision making for inventory management, marketing, and sales strategies.

## 🚀 Getting Started

1. **Clone the repository** to your local machine.
   ```bash
   git clone <repo-url>
   cd "Superstore Data Analysis"
   ```
2. **Open the data file** (`Superstore Data.xlsx` or similar) in Excel or another tool to review the raw dataset.
3. **Launch Power BI Desktop** and open the `Superstore Dashboard.pbix` file.
4. Use the built-in filters and slicers to interact with the visualizations and explore the data.
5. (Optional) Modify or extend the dashboard by importing updated data or creating new visuals.

## 📝 Notes

- The dataset is fictional and often used in training and demonstration scenarios.
- Power BI Desktop is required to open and modify the `.pbix` file; you can download it from [Microsoft Power BI](https://powerbi.microsoft.com/).
